## Shortcuts

| Keys  | Actions |
|-------|---------|
I | hide/show inspector window
P | hide/show palette window
Ctrl + S | save scene
Ctrl + O | load scene
Alt + S | save palette
Alt + O | load palette
Q | cancel insertion
Del | delete body/item/whatever when it selected
Enter | run/stop
